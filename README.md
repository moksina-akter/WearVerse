# WearVerse 👕✨

WearVerse is a modern fashion e-commerce frontend built with Next.js and Tailwind CSS. The project was developed as part of the OXIVOS Frontend Developer Round 1 Project Task.

## Live Demo

Live Site: https://wearverse-delta.vercel.app/

## GitHub Repository

Repository: https://github.com/your-username/wearverse

---

## Features

- Premium modern UI
- Responsive design for mobile, tablet and desktop
- Product listing using local dummy data
- Product details page
- Category filtering
- Search functionality
- Price sorting
- Add to Cart
- Wishlist functionality
- Cart quantity management
- Checkout flow
- Order success page
- Related products section
- About page
- Contact page
- FAQ section
- Newsletter subscription UI
- Empty states
- 404 Not Found page

---

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- Lucide React
- React Toastify
